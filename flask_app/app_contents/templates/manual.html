Text Message Budget Manual

Budget Actions

  • Add budget expense entry: bud category amount notes - ie. bud home 15 home depot
  • Add money to a budget: add category amount notes - ie. add home 25 bonus
  • Move money from one available budget to another: move amount from_category to_category - ie. move 50 brandon sheena
  • Modify monthy budget amount: mod category amount notes - ie. mod brandon 50 change from 40
  • Update categories: upd from_category to_category - ie. upd groceries food
  • Delete entry by number: del number - ie. del 1234

Budget Reports

  • This month summary report: sum
  • Category report: rep category ie. rep brandon - or just the category name

Text Message Maintenance Manual

Maintenance Actions

  • Add maintenance record: maintenance machine miles notes - ie. lexus 110000 oil change - if no miles ie. furnace 0 filter change

Text Message Garage Manual

Garage Actions

  • Check garage status: garage
  • Open garage: open garage_name - ie. open a
  • Close garage: clsoe garage_name - ie. close b
